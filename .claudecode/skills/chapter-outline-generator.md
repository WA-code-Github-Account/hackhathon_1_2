# Chapter Outline Generator

## Purpose
This skill generates detailed chapter outlines for books based on genre, theme, and target audience.

## Usage
Use this skill when you need to create a structured outline for a book project.

## Instructions
When generating a chapter outline:
1. Ask for the book's genre, theme, and target word count
2. Create 10-15 chapter titles with brief descriptions
3. Ensure logical flow and progression
4. Include plot points and character development notes
5. Suggest chapter word counts

## Example Output Format
```
Chapter 1: The Beginning (2000 words)
- Introduce protagonist
- Establish setting
- Present inciting incident

Chapter 2: The Call to Adventure (2500 words)
- Protagonist faces initial challenge
- Meet key supporting characters
- Introduce main conflict
```

## Best Practices
- Keep chapter lengths consistent
- Balance action and exposition
- Ensure each chapter ends with a hook
- Consider pacing throughout the book
